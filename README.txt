# Balls simulator
  - Simulate balls with interaction force 1/r - 1/r^2.
  - Simulation runs in its own thread.
  - Balls can walk trough each other.
  - Balls has an elastic reflection from the walls.
  - Balls can be created, moved or deleted.
  - Project doesn't include google test source code, but compiled google test is needed to compile the project (path to compiled google test is set by GTEST_ROOT variable in cmake).
